# Design Task Routing

| Task | Agent | Modell |
|------|-------|--------|
| UI/UX Design | frontend-ui-ux-engineer | `google/antigravity-gemini-3.1-pro` |
| Rapid Prototyping | frontend-ui-ux-engineer | `google/antigravity-gemini-3-flash` |
