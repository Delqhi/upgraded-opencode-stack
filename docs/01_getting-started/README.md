# Getting Started with OpenSIN-AI

## Quick Start

1. Clone: `git clone https://github.com/OpenSIN-Code/Infra-SIN-OpenCode-Stack.git`
2. Install: `./install.sh`
3. Create agent: `/create-a2a-sin-agent`

## Agent Config System v5

- `opencode.json` — Haupt-Config
- `oh-my-sin.json` — Team Register (20 teams)
- `oh-my-openagent.json` — Subagenten-Modelle
- `my-sin-team-*.json` — Team-spezifische Modelle

See [Agent Configuration Guide](https://github.com/OpenSIN-AI/OpenSIN-documentation/blob/main/docs/guide/agent-configuration.md)
